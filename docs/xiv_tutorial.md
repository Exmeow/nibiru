## 登录

**初次进入服务器**

```
/register <密码> <重复密码> 或 /reg <密码> <重复密码>
```

```
示例:/register 0123456 0123456 或 /reg 0123456 0123456
```

**之后进入服务器**

```
/login <密码> 或 /l <密码>
```

```
示例: /login 0123456 或 /l 0123456
```
