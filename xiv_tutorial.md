## 登录

### 初次进入服务器

```
/register <密码> <重复密码>
```

```
示例:/register 0123456 0123456
```

###  之后进入服务器

```
/login <密码>
```

```
示例:/login 0123456
```

### CoreProtect

> 查询方块放置/破坏和容器存入取出记录。

```
/co i
```
